# Sistema de Gestión de Restaurante El Puerto de la Costa

## Estructura del proyecto
- backend/server.js - Servidor Node.js en puerto 3001
- frontend/index.html - Interfaz principal del sistema
- frontend/menu-cliente.html - Menú QR para clientes
- frontend/servidor.py - Servidor Python en puerto 3000
- package.json - Dependencias del proyecto

## Tecnologías utilizadas
- Node.js + Express
- MySQL (XAMPP)
- Python
- HTML/CSS/JavaScript
